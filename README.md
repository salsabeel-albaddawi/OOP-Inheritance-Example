# Java Inheritance Example

This project demonstrates basic inheritance in Java.

## Classes
- **A**: Parent class
- **B**: Child class that extends A and overrides method `m3`
- **Test**: Contains the `main` method to test the classes

## Concepts Used
- Inheritance
- Method Overriding
- `super` keyword
